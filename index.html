About

Superficie minima is a tool for studying cohomological invariants of minimal algebraic surfaces over the complex numbers. It was designed because I did not have a grasp on how these invariants were related, and how several theoretical inequalities influence the range of possible values.

Contact

For contact details, please see my homepage. Please report bugs; if you think of a feature, I would like to hear it as well. Any tips about presentation and layout are welcome. If you have data for the bibliography/reference/theory block about a (class of) surface(s), you can send it to me via email.
